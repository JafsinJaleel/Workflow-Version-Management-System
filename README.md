Workflow Version Management System
Overview

This project implements a Workflow Version Management System using Flask and MySQL.
The system allows users to create workflows with multiple stages, update workflows, maintain version history, and compare differences between workflow versions.

The application automatically creates a new version whenever a workflow is updated, enabling tracking of workflow evolution over time.

Technologies Used:
- Python Flask
- MySQL
- HTML/CSS/JavaScript

Features:
- Create workflows
- Version control
- Version history
- Version comparison
- Edit workflows
- Delete workflows
